# chat-tp

## Description

AI-enhanced real-time chat

## Functionalities

1.  Communication En Temps Réel
2.  Traduction Automatique
3.  Validation De L'Information -> hover sur le badge donne la raison
4.  Suggestions De Réponse
5.  Speech to text

## Installation

### chat-api

```bash
cp .env.exemple .env # Copy and config .env
pnpm install
```

### chat-front

```bash
cp .env.exemple .env # Copy and config .env
pnpm install
```

## Running the app

### chat-api

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

### chat-front

```bash
# development
$ npm run dev

# production mode
$ npm run start
```
